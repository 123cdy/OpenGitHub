# OpenGitHub
一款基于Material Design + Kotlin + MVVM + ViewModel + LiveData  + Room + Retrofit + Okhttp + 协程 + LiveDataBus + Glide等架构实现Github客户端项目(持续更新中),欢迎start or fork
